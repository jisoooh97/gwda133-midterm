I learned Css

Css stands for, cascading style sheet and we need it in out html to add style into our html such as backgroundcolor, text-align,font, spacing and etc.

CSS Box Model

Box Model is what wraps the html page: margin, border, padding and content. A html can be considered as a box and it is used when designing and layout. TRBL stands for top, right, bottom, and left. TRBL is used in Box Model for sequence and this is important because it is important where you exactly want to place your box.
